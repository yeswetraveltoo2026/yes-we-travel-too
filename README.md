# Images Folder

Upload all your blog and page photos here.

## How to name your image files
Use simple, lowercase names with hyphens — no spaces:
- ✅ `nairobi-market.jpg`
- ✅ `lisbon-tiles.jpg`
- ❌ `Nairobi Market Photo.jpg`

## How to use an image in a post
In your post HTML file, reference the image like this:

```html
<img src="images/nairobi-market.jpg" alt="A busy market in Nairobi, Kenya" class="post-img"/>
```

## Recommended image sizes
- **Hero / cover photos**: 1200×675px (16:9 ratio) — keep under 500KB
- **In-post photos**: 900×600px — keep under 300KB

## Free tools to resize/compress images before uploading
- squoosh.app (free, browser-based)
- tinyjpg.com (free compression)
